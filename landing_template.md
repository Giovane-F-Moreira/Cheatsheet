Landing Page Template Cheatsheet
===============

#### How to setup the template?

#####Step1: Go to the Main Directory
```
cd /Desktop/Harrison/LandingPage-Hack/Referral_Landing-master
```

#####Step2: Install Libraries
```
npm bower install
```

#####Step3: Start the Server
```
npm start
```

#### How to change the content?

#####Background Image
```
--------------------------------------------------------
In Folder
--------------------------------------------------------
1. Go to /app/assets/img/background
2. Find the crossponding background and open with "PHOTOSHOP"
3. Open the background you choose with "PHOTOSHOP"
--------------------------------------------------------
In Photoshop
--------------------------------------------------------
4. cmd + option + c //see the size of the canvas of template image
5. cmd + option + i //change the size of the image we choose
6. Copy the "image" we choose and paste into the "canvas"
7. Adjust the image to fit in the canvas

// Changing features images, footer images...etc are the same
```
